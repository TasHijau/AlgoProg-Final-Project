# Interactive Data Visualization Dashboard

## Overview
This project is a Semester 1 Algorithm and Programming assignment.
It is a Python GUI program that allows users to upload CSV files
and display data using line graphs.

The program helps users understand data more easily by using
visual graphs instead of text output.

## How to Run
1. Make sure Python is installed.
2. Open a terminal in the project folder.
3. Run this command:

python gui.py

4. Upload a CSV file and select the options to generate a graph.

## Dependencies
- Python 3
- pandas
- matplotlib
- tkinter

Install required libraries using:

pip install pandas matplotlib

Note: Tkinter is included with most Python installations.
